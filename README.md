# 职场红领巾--小程序

## 使用框架 wepy

### 介绍
 * github：[wepy](https://github.com/Tencent/wepy)
 * 官网： [小程序组件化开发框架](https://tencent.github.io/wepy/)

### 使用

  * 克隆到本地
  ```
  git clone git@github.com:careerbro/zchlj-weixin-app.git
  ```
  * 安装依赖包
  ```
  npm install (npm)
  yarn (yarn)
  ```
  * 运行程序
  ```
  wepy build --watch
  ```
  * 使用微信开发者工具打开项目下 ／dist 文件夹
